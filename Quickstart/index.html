Some simple, basic things to get you started hacking some AppleTV apps. Most of these files will need some editing to work properly for you (server URLs, etc.).

  • addsite.mc - Adds the "Add Site" application to the main AppleTV Screen
  • feature.mc - Adds some hidden features: EnableFeatureEnabler, PrintBitRate, Home Sharing diagnostics
  • mitm.mc - Sets up a WiFi network with a proxy, suitable for intercepting traffic (like Burp suite or mitmproxy).
  • bag.plist.xml - Basic AppleTV app configuration file. You'll need to rename it just bag.plist (and move the other one out of the way) to get this to load on the AppleTV).
  • bag.plist - A real bag.plist for the original AppleTV Trailers app (fetched from http://trailers.apple.com/appletv/bag.plist)
  • debug.log - The debug log that gets collected when the AddSite profile is installed (and configured to point at this server).
